# Computed Schur-Weingarten Degree Tables

These CSV files were generated by

```bash
julia generate_degree_tables.jl
```

from the row and column signatures used in the existing comparison tables under
`../intersection_degree_computations_and_parametrizations/degree_tables`.

The entries are exact integer degrees computed by
`degree_TT_variety(D, d; method=:schur_weingarten)`.

The generator writes `0` for signatures that are outside the TT admissibility
range used by the Schur-Weingarten parametrization.
